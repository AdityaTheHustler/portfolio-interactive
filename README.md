## Overview

This portfolio breaks the traditional mold by implementing a gamified hiring process. Visitors must complete challenges (Snake or Pong) to unlock professional contact information.

### Key Features

- **Gamified Hiring Protocol**: Play Snake or Pong to prove your determination
- **Matrix Theme**: Retro hacker aesthetic with green scanlines and CRT effects
- **Interactive Terminal**: Functional command-line interface
- **Screen Glitch Effects**: High-quality tech glitches on interactions
- **Responsive Design**: Optimized for all devices
- **No Framework Dependencies**: Pure vanilla JavaScript and Tailwind CSS

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Tailwind CSS (CDN)
- **Games**: HTML5 Canvas API
- **Animations**: CSS Keyframes, RequestAnimationFrame
- **SEO**: Open Graph, Twitter Cards, JSON-LD Structured Data

## Games

### Snake Xenzia
- Classic Snake with wall-wrapping mechanics
- Dynamic speed increase
- Bonus food with timer (50 points)
- Win condition: 400 points

### Cyber Pong
- AI-powered opponent
- Smooth paddle controls
- Best of 4 rounds
- Radius-based collision detection
